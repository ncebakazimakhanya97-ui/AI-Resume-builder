# AI Resume Builder

An AI-powered resume builder that generates professional resumes based on user input. The application helps job seekers create ATS-friendly resumes quickly and efficiently.

## Features
- AI-generated resume content
- Multiple resume templates
- User-friendly interface
- Downloadable resumes

## Tech Stack
- HTML, CSS, JavaScript
- Python (AI content generation)
- Streamlit / Web deployment
